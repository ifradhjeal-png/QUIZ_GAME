
# Quiz Score Tracker (Java)

## About this project

This is a Java program I made to practice file handling, arrays and random questions.

The program asks questions to students, checks their answers and gives them a score.

---

## How the program works

* First, the user enters number of students
* Then each student enters their name
* Each student gets 4 random questions
* The same question will not repeat
* If the answer is correct:

  * A random score between 1 and 10 is given
* If wrong:

  * No score is given
* At the end:

  * Total score is calculated
  * Average score is also calculated

---

## Files needed

These files must be in the same folder:

* QUIZ_QUESTIONS.txt → contains questions
* QUIZ_QUESTIONS_ANSWERS.txt → contains answers

Important:

* Questions and answers must be in the same order
